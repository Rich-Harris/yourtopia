Install
=======

Grab the source using mercurial from http://bitbucket.org/rgrp/worldtopia::

    hg clone http://bitbucket.org/rgrp/worldtopia

Install the requirements (in a virtualenv)::

    # create a virtualenv
    virtualenv ../pyenv-worldtopia
    # install the app requirements
    pip -E ../pyenv-worldtopia install -r pip-requirements.txt

Run the app::

    python worldtopia/app.py
